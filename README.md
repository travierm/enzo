# Enzo

### Features
- JSX templating
- Preact component hydration
- MVC folder structure

### Tech Stack
- Hono for routing
- Preact as a JSX templating engine
- HTMX for frontend interactivity
- Tailwind for styling
- Running on Bun

### How to use
Install deps:

```bash
bun install
```

To run:

```bash
bun dev
```
